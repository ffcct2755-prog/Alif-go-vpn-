# আলিপ গো ভিপিএন সার্ভার ব্যাকএন্ড (Alif Go VPN Server Backend API)

এই ফোল্ডারে আপনার ভিপিএন অ্যাপের জন্য একটি সম্পূর্ণ ডাইনামিক **PHP + MySQL** ব্যাকএন্ড কোড রয়েছে। এটি আপনার হোস্টিংয়ে আপলোড করে খুব সহজেই সার্ভার লিস্ট রিমোটলি কন্ট্রোল করতে পারবেন।

## ব্যাকএন্ডে কি কি আছে?
1. **`database.sql`** - ডাটাবেজ টেবিল এবং ডেমো সার্ভার লিস্ট।
2. **`db_connect.php`** - ডাটাবেজের সাথে সংযোগের ফাইল।
3. **`get_vpn_servers.php`** - ভিপিএন অ্যাপের জন্য JSON API।
4. **`add_server.php`** - নতুন সার্ভার অ্যাড করার জন্য API।
5. **`delete_server.php`** - সার্ভার ডিলিট করার জন্য API।
6. **`index.php`** - একটি অত্যন্ত সুন্দর ও রেসপনসিভ **ওয়েব অ্যাডমিন কন্ট্রোল প্যানেল** যার মাধ্যমে যেকোনো মোবাইল বা পিসি থেকে সার্ভার লিস্ট লাইভ কন্ট্রোল করা যাবে!

---

## কিভাবে সেটআপ করবেন?

### ১. ডাটাবেজ তৈরি করুন
* আপনার হোস্টিংয়ের cPanel বা প্যানেলে লগইন করে একটি নতুন **MySQL Database** এবং **Database User** তৈরি করুন।
* তৈরি করা ইউজারকে ডাটাবেজের সব পারমিশন (Privileges) দিন।
* **phpMyAdmin** এ যান এবং `database.sql` ফাইলটি **Import** করুন। এটি স্বয়ংক্রিয়ভাবে `vpn_servers` টেবিল তৈরি করবে এবং কিছু ডেমো ডেটা যোগ করবে।

### ২. ডাটাবেজ কানেকশন আপডেট করুন
* **`db_connect.php`** ফাইলটি এডিট করুন এবং নিচের তথ্যগুলো বসান:
  ```php
  $host = "localhost";
  $username = "আপনার_ডাটাবেজ_ইউজারনেম";
  $password = "আপনার_ডাটাবেজ_পাসওয়ার্ড";
  $dbname = "আপনার_ডাটাবেজ_নাম";
  ```

### ৩. ফাইল আপলোড করুন
* এই ফোল্ডারের সব PHP এবং SQL ফাইলগুলো আপনার সার্ভারের (যেমন: `alifgo.gt.tc/api/` বা আপনার ডোমেইন) একটি ফোল্ডারে আপলোড করে দিন।

### ৪. অ্যাপের সাথে লিংক করুন
* আপনার অ্যাপের অ্যাডমিন প্যানেলে যান এবং **"Sync Servers from JSON URL"** ফিল্ডে আপনার `get_vpn_servers.php` এর Raw URLটি বসান।
  * উদাহরণ: `https://alifgo.gt.tc/api/get_vpn_servers.php`
* **Sync Servers Now** বাটনে ক্লিক করুন। অ্যাপটি সাথে সাথে আপনার লাইভ ব্যাকএন্ড ডাটাবেজ থেকে সার্ভারগুলো সিঙ্ক করে নিবে!
